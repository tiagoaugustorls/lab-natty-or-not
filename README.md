# Vídeo: I'll be Back 🎥
## 📒 Descrição
Este vídeo de 3 segundos apresenta uma avatar de minha pessoa falando a famosa frase de O Exterminador do Futuro 2.

## 🤖 Tecnologias Utilizadas
IA Generativa Gemini para criação da foto usada no avatar;
IA Generativa D-ID para sintetização de voz e geração de avatar virtual;

## 🧐 Processo de Criação
Usei o "gemini" para criar um avatar em forma de desenho. O "D-ID" foi essencial para criar um avatar virtual e voz sintetizada, trazendo um movimento.

## 🚀 Resultados
o vídeo é somente eu em forma de desenho falando i'll be back.
